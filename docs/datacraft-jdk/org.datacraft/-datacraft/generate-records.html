@JvmStatic
fun <T> generateRecords(json: String, iterations: Long, type: Class<T>): Iterator<T>

Generates an iterator over instances of a specified type, populated with data according to the field specifications.

This method uses Gson to serialize each map into an instance of the specified class type. This method is useful when you need to produce data records as instances of a data class or other class types.

Return

an iterator over instances of the specified type.

Parameters

iterations

the number of records to generate.

type

the class of the type to which the generated records should be converted.

Throws

JsonSyntaxException

if JSON serialization or deserialization fails.
